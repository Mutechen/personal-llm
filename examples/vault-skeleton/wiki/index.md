# Wiki index

*This is your agent-maintained knowledge base. As you ingest source material into `raw/` and chat with the agent, this wiki grows. The agent updates `wiki/` files during sleep-time (Phase 1+).*

For Phase 0 this is just a placeholder — the wiki population logic lands in Phase 1.

## Sections

- `daily/` — daily summaries
- `topics/` — topic-organized notes
- `projects/` — project-scoped notes
- `projects/learning/` — active learning goals
- `imported/` — knowledge imported from `.lobe` files
