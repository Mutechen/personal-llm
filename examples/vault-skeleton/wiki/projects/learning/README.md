# Active learning goals

*Drop a markdown file in here for each topic you want the agent to learn over time. The agent picks goals from this directory during sleep-time learning sessions (Phase 2+).*

Format (one file per goal):

```markdown
# Goal: <topic>

## Why I care
(your reasons)

## What "good" looks like
(what would satisfy you)

## Sources I trust
(books, sites, papers)

## Don't
(what the agent should avoid researching)
```

The agent updates each goal file with progress notes and links to wiki pages it built.
